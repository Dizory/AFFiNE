## 🎯 Готово к использованию

### Запуск (разработка)

```bash
cd /workspace/packages/frontend/apps/tauri
yarn install
yarn dev
```

### Сборка (production)

```bash
cd /workspace/packages/frontend/apps/tauri
yarn build
```

Результат в `src-tauri/target/release/bundle/`
